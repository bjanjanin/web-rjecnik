A

Accessibility (Pristupačnost)

Accessibility (pristupačnost) se odnosi na tehnike prilagođavanja stranice osobama sa posebnim potrebama. Kada je stranica pristipačna (accessible), svaki korisnik može pristupiti sadržaju na web stranici, čak i kada ima problema sa primjerice vidom ili sluhom, ili je pak primoran za navigaciju koristiti isključivo tipkovnicu.

Accesskey

Accesskeys, odnosno tipke za pristupačnost, se koriste u tipkovničkim kraticama kako bi korisnik brzo došao do određenog dijela stranice.

AJAX

AJAX je kratica za Asynchronous JavaScript and XML i predstavlja tehniku koja kombinira razne tehnologije za izradu interaktivnih aplikacija.

Aplikacija

Aplikacija je sinonim za program. Ona može biti native (namijenjena za instaliranje na uređaj) ili na webu (pregledavate ju ako imate pristup internetu). Aplikacijom najčešće zovemo softver koji preuzmemo i koristimo na mobilnim uređajima, no sve se češće koristi i za web softver i softver na Mac računalima.

B

Best Practice

Pojam best practice se odnosi na način izrade i tehnologije koje se od struke smatra preporučenima, odnosno najboljima. Najboljim se praksama primjerice smatra korištenje web standarada, responzivnog dizajna, mobile first pristupa, itd.

Box Model

Box Model (model kutije) u CSS-u se odnosi na model po kojem se elementi stranice formatiraju. Svaki element je dvodimenzionalni četverokutni objekt, odnosno svojevrsna kutija. Model te kutije čine padding, margin i border.

Browser

Browser ili web preglednik je aplikacija za pregledavanje web stranica. Neki od popularnih preglednika su Chrome, Firefox i Internet Explorer.

C

Chrome

Google Chrome je Jedan od najkorištenijih web preglednika. Pojam chrome se često koristi i za dio korisničkog sučelja koji čini okvir prozora. U tom se dijelu obično nalaze razne tipke, polja, kartice (tabovi), itd.

Client-side

Pojam client-side se odnosi na tehnologije koje koristi i probavlja web preglednik. Primjeri takvih tehnologija su HTML, CSS i JavaScript. Client-side developerima se nazivaju osobe koje se tim tehnologijama služe prilikom izrade stranica.

CMS

Content Management System je sustav za održavanje web stranica. Jedan od najpopularnijih CMS-ova je Wordpress.

CSS

Cascading Style Sheets je jezik koji se koristi za definiranje izgleda web stranica.

D

Div

Div je jedan od elemenata u HTML jeziku, te je skraćenica engleske riječi divison (podjela). Ovaj HTML element se koristi za grupiranje sadržaja.

Doctype

Doctype (Document Type Declaration) je naredba na vrhu HTML dokumenta koja pregledniku govori koju verzijom HTML-a se stranica služi.

F

Firefox

Firefox je jedan od najpopularnijih web preglednika. Proizvodi ga tvrtka Mozilla.

Flash

Multimedijski alat baziran na vektorskoj tehnologiji. Često se koristi na web stranicama, najčešće u kombinaciji s drugim tehnologijama. Sve se rjeđe upotrebljava za izradu dijelova web stranice, s obzirom na brojne nedostatke (poput poteškoća kod indeksiranja od strane tražilica i manjka fleksibilnosti).

Font

Font je dizajnirana skupina slova. Jedan ili više fontova čine typeface, odnosno pismo. Primjerice, pismo bi bilo Helvetica, dok bi font bila jedna varijanta tog pisma, poput Helvetica Bold.

FTP (File Transfer Protocol)

FTP je metoda za prebacivanje datoteka na server. Često se koristi kao metoda prebacivanja stranica na internet (upload).

H

Hosting

Hosting je smještaj web stranica na serveru ili hostu. Zakup prostora na serveru se najčešće plaća pružatelju hosting usluga.

HTML

HTML je kratica za HyperText Markup Language, jezik kojim se izrađuju web stranice. HTML-om se formatira sadržaj web stranica.

Hyperlink

Hyperlink (ili kraće link), je poveznica s jedne na drugu web stranicu. Link je najčešće dio teksta ili grafike.

I

Information Architecture (informacijska arhitektura)

Informacijska arhitektura bavi se organizacijom sadržaja na web stranicama.

Internet Explorer

IE je jedan od popularnijih web preglednika za Windows računala. Proizvodi ga tvrtka Microsoft.

J

JavaScript

JavaScript je popularni jezik, koji se koristi za izradu web stranica.

JPEG

JPEG (.jpeg, .jpg) je grafički format koji se koristi za sažimanje fotografija i raznih grafika za web stranice.

jQuery

jQuery je jedna od najpopularnijih JavaScript biblioteka.

M

Mobile First

Pojam Mobile First se odnosi na process dizajniranja koji kreće od najmanjih rezolucija, prema većima. Na taj se način, s obzirom na veličinu ekrana, prvo posvećujemo najbitnijim informacijama.

Mockup

Mockup je napredna skica web stranice. Obično se izradi kao statički ili interaktivni predložak i služi kao korak prema sljedećoj fazi izrade.

MySQL

MySQL je najpopularniji sistem za upravljanje bazama podataka.

O

Opera

Opera je jedan od najkorištenijih web preglednika.

OS X

OS X je deseta verzija operativnog sistema za Apple računala (X se izgovara ten).

P

PHP

PHP je popularni jezik koji se izvodi na serveru a može koristiti u HTML-u.

PNG

PNG (.png) je jedan od najpopularnijih grafičkih formata, koji se danas na stranicama koristi kao zamjena za GIF (.gif). Glavna odlika PNG formata je što podržava varijabilnu transparentnost.

R

Readability (čitljivost)

Readability je lakoća čitanja i razumijevanja teksta. Kako bi tekst bio čitljiviji, moramo odabrati odgovarajući font, kontrast boja, veličinu, itd.

Responzivni web dizajn

Responsizni web dizajn podrazumijeva korištenje tehnika i tehnologija za prilagodbu web stranica uređaju, odnosno veličini ekrana. Responzivne stranice će se tako prilagoditi i malim ekranima mobilnog telefona i velikim ekranima stolnih računala.

S

Safari

Safari je jedan od najpopularnijih web preglednika. Koristi jednaki softver (tzv. engine) kao i Chrome, što znači da će stranice u ova dva preglednika izgledati jednako.

Search Engine Optimization

Search Engine Optimization (skraćeno SEO) je pojam koji se odnosi na optimizaciju stranica za tražilice. Bolje optimizirane stranice se mogu lakše pronaći, odnosno više su pozicionirane u rezultatima pretraživanja.

S

Server-side

Pojam server-side odnosi se na tehnlogije koje koristi server, kao što je primjerice PHP.

T

Typeface (pismo)

Typeface je dizajnirana skupina slova, odnosno pismo. Sastoji se od jednog ili više fontova. Popularni primjeri pisma na WWW-u su Georgia, Verdana i Times New Roman. Georgia pismo se primjerice sastoji od više fontova, kao što su Georgia Bold i Georgia Italic.

U

Usability

Usability možemo prevesti kao iskoristivost web stranice. Ako je iskoristivost web stranice velika, tada je možemo koristiti s velikom lakoćom. Na iskoristivost utječe mnogo faktora, kao što su tipografija, dizajn stranice, informacijska arhitektura, itd..

User Interface (korisnično sučelje)

Korisničko sučelje je ono što korisnik vidi i čime se služi na web stranici.

V

Validation (validacija)

Validacija je proces provjeravanja ispravnosti HTML ili CSS koda.

W

Web standardi

Web standardi su skupina pravila za izradu web stranica.
