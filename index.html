A

Accessibility

Pojam accessibility (pristupačnost) se odnosi na tehnike prilagođavanja stranice osobama sa posebnim potrebama. Kada je stranica pristipačna (accessible), svaki korisnik može pristupiti sadržaju na web stranici, čak i kada ima problema sa primjerice vidom ili sluhom, ili je pak primoran za navigaciju koristiti isključivo tipkovnicu.

Accesskey

Accesskeys, odnosno tipke za pristupačnost se koriste u tipkovničkim kraticama kako bi korisnik brzo došao do određenog dijela stranice.

AJAX

AJAX je kratica za Asynchronous JavaScript and XML i predstavlja tehniku koja kombinira razne tehnologije za izradu interaktivnih aplikacija.

B

Box Model — model kutije

Box Model (model kutije) u CSS-u se odnosi na model po kojem se elementi stranice formatiraju. Svaki element je dvodimenzionalni četverokutni objekt, odnosno svojevrsna kutija. Model te kutije čine padding, margin i border.

Browser

Browser ili web preglednik je aplikacija za pregledavanje web stranica. Neki od popularnih preglednika su Chrome, Firefox i Internet Explorer.

C

Chrome

Google Chrome je Jedan od najkorištenijih web preglednika. Pojam chrome se često koristi i za dio korisničkog sučelja koji čini okvir prozora.

Client-side

Client-side tehnologije su tehnologije koje koristi i probavlja web preglednik. Primjeri takvih tehnologija su HTML, CSS i JavaScript.

CMS

Content Management System je sustav za održavanje web stranica. Jedan od najpopularnijih CMS-ova je Wordpress.
